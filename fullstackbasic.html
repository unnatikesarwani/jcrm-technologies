Syllabus

Full Stack Development — Course Modules

Structured modules to take you from the basics of web to building a working full-stack project using PHP & MySQL (beginner-friendly).

Module 1: Introduction to Web Development
  • • Understanding how the web works (client-server model)
  • • Overview of frontend and backend development
  • • Setting up the development environment (VS Code, XAMPP/WAMP, Git)
  • • Basic file structure for web projects
Module 2: Frontend Development (Beginner Level)
HTML (Structure of Web Pages)
  • • Basic tags and attributes
  • • Forms and inputs
  • • Semantic HTML
  • • Tables, lists, and media
CSS (Styling Web Pages)
  • • Selectors and properties
  • • Colors, typography, and backgrounds
  • • Box model, margins, padding
  • • Layouts: Flexbox and Grid
JavaScript (Basics of Programming for the Web)
  • • Variables, data types, operators
  • • Functions and scope
  • • DOM manipulation
  • • Events and event handling
Bootstrap (Responsive Design)
  • • Grid system
  • • Components (navbar, cards, buttons, forms)
  • • Utilities and customization
Module 3: Backend Development (Beginner Level)
PHP (Server-Side Scripting)
  • • Introduction to PHP syntax
  • • Variables, arrays, loops, functions
  • • Handling forms with PHP
  • • Sessions and cookies
SQL (Databases)
  • • Introduction to relational databases
  • • Creating databases and tables
  • • CRUD operations (Create, Read, Update, Delete)
  • • Joins and relationships
Connecting PHP with SQL
  • • PHP-MySQL connection
  • • Performing CRUD with PHP and MySQL
  • • Handling errors and validations
Module 4: Live Project 1 (Beginner Level)
Mini Project Examples:
  • • Personal Portfolio Website
  • • Contact Form with PHP + MySQL storage
  • • Simple Blog System
Extras

Hands-on Labs & Assessments

Each module includes hands-on lab tasks, small quizzes and a final mini-project to demonstrate CRUD & routing concepts.
